# Archive Directory

This directory contains deprecated, superseded, and legacy format documents from the Anti-Hate Platform project. All files here are preserved for historical reference but are no longer active in the current project structure.

## Structure

### `v0.0.1/Architecture/`
Contains early versions of the technical roadmap that have been superseded by newer iterations.

**Contents:**
- `Anti-Hate_Platform_Technical_RoadMap_version0.01_OLD.md` - The original roadmap draft
- `Anti-Hate_Platform_Technichal_RoadMap_Evolution.md` - Evolution notes superseded by v0.0.2

**Current Version:** See [`/Architecture/Anti-Hate_Pleatform_Technical_RoadMap_v_0_0_2.md`](../Architecture/Anti-Hate_Pleatform_Technical_RoadMap_v_0_0_2.md)

---

### `Legacy_Formats/Requirements/`
Contains binary format versions (`.docx`, `.pdf`) of the requirements documentation. These are duplicates of the canonical markdown versions.

**Contents:**
- `Functional Requirements Specification.docx`
- `Functional Requirements Specification.pdf`
- `Non-Functional Requirements Specification.docx`
- `Non-Functional Requirements Specification.pdf`

**Current Versions (Markdown):**
- [`/Requirements/Functional Requirements Specification.md`](../Requirements/Functional%20Requirements%20Specification.md)
- [`/Requirements/Non-Functional Requirements Specification.md`](../Requirements/Non-Functional%20Requirements%20Specification.md)

---

### `Superseded/Charter_Drafts/`
Contains early draft versions of the Charter in ODT and PDF formats. These have been superseded by the current Governance package.

**Contents:**
- `Charter of the Global Anti-Hate Speech Platform (The Anti-Hate Platform).odt`
- `Charter of the Global Anti-Hate Speech Platform (The Anti-Hate Platform).pdf`
- `Can Anti-Hate Platform Leverage from Wikipedia.odt`

**Current Version:** See [`/Governance/Anti-Hate_Platform_Charter.md`](../Governance/Anti-Hate_Platform_Charter.md)
